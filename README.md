# 🏥 Hopital-Gestion

## 📌 Description
Hopital-Gestion est une application en C/C++ conçue pour gérer les opérations essentielles d’un hôpital.  
Elle permet de suivre les patients, le personnel médical et les dossiers administratifs de manière centralisée.

---

## 🚀 Fonctionnalités
- Gestion des patients (ajout, modification, suppression, suivi médical).  
- Gestion du personnel hospitalier (médecins, infirmiers, administrateurs).  
- Gestion des services hospitaliers (consultations, hospitalisations, urgences).  
- Gestion des dossiers médicaux via fichiers CSV et base de données.  
 
---

## 🛠️ Technologies utilisées
- **Langages** : C, C++  
- **Base de données** : Fichiers CSV (`Dossier_Med.csv`) et SQLite (`Gestionnaire.db`)  
- **IDE/Build** : Code::Blocks (`.cbp`, `.layout`, `.depend`)  

---

## 📂 Installation et utilisation
1. Cloner le dépôt :
   ```bash
   git clone https://github.com/RAYANEBIO/Hopital-Gestion.git
   ```
2. Ouvrir le projet dans **Code::Blocks** ou un autre IDE compatible.  
3. Compiler le projet (`main.cpp` et `DatabasManager.c`).  
4. Lancer l’application pour gérer les patients et le personnel hospitalier.
